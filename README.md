# DecodeLab Internship Support Chatbot 

## About The Project
This project was built as part of Decode Lab AI Internship Bootcamp 2026.  
It's a console-based Python chatbot that helps interns with program details, tasks, attendance, certificates, mentor info, and post-internship job opportunities.

## Key Features
- Internship Guidance: Program details, 6 tracks, and batch info
- Task Submission: Submit projects with timestamps
- Status Check: Check application status using Intern ID
- Attendance Tracker: Calculate attendance % for certificate eligibility
- FAQ System: Answers to 5 most common intern questions
- Human Support: Ticket system for contacting support team
- Career Module: Remote job opportunities with eligibility check
- Feedback System: Collect intern feedback and ratings

## Technologies Used
`Python 3` | `Regex` | `Datetime` | `Console I/O`

## How It Works
1. User enters Name, Email, Phone, and Intern ID
2. Select options from menu 1-11
3. Get help with tasks, attendance, certificates, and jobs
