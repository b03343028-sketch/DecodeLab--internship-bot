# AI Recommendation System

A console-based Python project that recommends items to users based on their interests, ratings, likes, and history.

## Features
- User Registration and Login
- Browse Catalog, Search by Category, Tag, Rating
- Get Personalized Recommendations + Top 10
- Rate Items, Like/Dislike, Add to Favorites
- View History, Interests, Trending Items
- Statistics and Export to CSV
- Data saved in `catalog.json` and `profiles.json`

## How to Run
1. Open in Google Colab or Jupyter
2. Run all cells in `Copy_of_recomandation_system_py.ipynb`
3. Follow the menu options
