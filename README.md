# DecodeLab Internship Support Chatbot

A rule-based console chatbot built for **DecodeLabs Industrial Training Kit** interns.  
This bot helps interns with program info, task submission, attendance, certificates, mentor details, FAQs, support tickets, and post-internship job opportunities.

## 🚀 Features
1. **Internship Program Details** - View available tracks and duration
2. **Application Status Check** - Check if application is Accepted or Under Review  
3. **Task / Project Submission** - Submit tasks with filename and notes
4. **Mentor & Batch Information** - Get mentor name and session timings
5. **Certificate Information** - Check certificate requirements and eligibility
6. **Attendance Calculator** - Calculate % attendance and check 80% requirement
7. **FAQ Section** - Answers to common internship questions
8. **Human Support Ticket** - Create support tickets for issues
9. **Remote Job Opportunities** - Apply for post-internship remote roles
10. **Feedback System** - Rate and review your internship experience
11. **Exit** - Close the chatbot

## 🛠️ How to Run
1. Open `decodelab_internship_bot.ipynb` in Google Colab
2. Click `Runtime > Run all`
